# 🛍️ Nostra – Modern Fashion E-Commerce Website

A fully responsive fashion e-commerce website built using **HTML5**, **CSS3**, and **JavaScript**. Nostra offers a clean, modern shopping experience with responsive layouts, product filtering, search functionality, interactive UI elements, and elegant animations.
